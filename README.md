# Expense Tracker

## Description
Expense Tracker is a Python-based application that helps users manage their daily expenses. Users can add, view, calculate and delete expenses. Data is stored permanently using CSV files.

## Features

- Add new expenses
- View all expenses
- Calculate total expenses
- Delete expenses
- Input validation
- CSV data storage

## Technologies Used

- Python
- CSV Module
- DateTime Module

## How to Run

1. Download the project
2. Open terminal in project folder
3. Run:

python main.py
